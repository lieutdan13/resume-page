Turns any Wordpress page into a beautiful Resume Page with built in Gihub activity and integrated lightbox portfolio.

== Description ==
Resume Page does one thing; it turns any page into a Resume Page. It has all the built in basics for what a resume needs, and more! It follows the tradtional resume layout, and is completely customizable with built in design options and an area for custom CSS. You could turn every page of your site into a different resume page. It also has a smart templating feature that let's developers in the house easily override the template. Just drop it into your child theme, and mod away. 

It's just that simple.

Features
* Reusable - use it on every page of your site and each can be different
* Github - integrated Github activity stream
* Portfolio - integrated lightbox portfolio
* Templating - themers can copy the template from /templates/resume-page-template.php into a child theme and mod away safely
* Themes - choose from a flat setup or stacked paper corners with CSS3


== Installation ==
1. Upload "resume-page" folder to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Activate Resume Page on any Wordpress page
4. Follow the instructions on the Resume Page Panel, and fill out any information
